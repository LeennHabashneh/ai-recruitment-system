# ai-recruitment-system
AI-based recruitment screening system using TF-IDF, graph mining, and Flask
## Overview
This project is a web-based recruitment screening system built using Flask. It analyzes candidate CVs and ranks them based on job descriptions using machine learning and graph analysis techniques.

## Features
- CV parsing and text extraction (PDF)
- TF-IDF similarity matching between CVs and job descriptions
- Candidate ranking system
- Graph-based candidate analysis using NetworkX
- Centrality metrics (PageRank, Degree, Betweenness, Closeness)
- Evidence-based chatbot for recruitment insights

## Technologies Used
- Python
- Flask
- Scikit-learn
- NetworkX
- Pandas & NumPy

## How It Works
1. Upload job description and CVs
2. System processes and cleans text
3. TF-IDF calculates similarity scores
4. Graph analysis identifies strong candidates
5. Results are displayed with rankings and insights
